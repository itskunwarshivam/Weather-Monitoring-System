# Weather Monitoring System

## Project Description
The Weather Monitoring System is a simple Java console-based application developed using Core Java and Object-Oriented Programming concepts. This project accepts temperature and humidity values from the user, analyzes the weather conditions, and displays appropriate alerts based on predefined thresholds.

## Features
- Accepts temperature input from the user
- Accepts humidity input from the user
- Detects high temperature conditions
- Detects low temperature conditions
- Detects high humidity conditions
- Displays a complete weather report
- Uses OOP concepts such as classes, objects, constructors, methods, and encapsulation

## Technologies Used
- Java
- Core Java
- OOP Concepts
- Scanner Class
